# Intelligent-Sales-Prediction-System-
Intelligent Sales Prediction System is a Data Science and Machine Learning project built with Python and Streamlit. It leverages EDA, Linear Regression, Random Forest Regressor, and Decision Tree Classifier to analyze sales data, predict future trends, forecast sales, identify return behavior, and generate actionable business recommendations.

Note: Refer the online sales dataset from kaggle , save it as "online sales datasett" and make sure that dataset and code remain on the same folder.
